DEAP Command Line

The command line is a slightly modified version of the wxPython shell, which is a Python interpreter. The command line is an interactive vehicle for plotting and anything else that you desire. All installed Python modules are available from the command line. Additionally, the command line is augmented with several functions mentioned below to support plotting.

More information about Python and its syntax is available at the Python website.

draw()

    Clears the plotting canvas.

draw()

    Generates a redraw event, which refreshes the plot.

export(file)

    Allows the user to save the figure displayed in the plotting panel to a file. Supported graphical formats are: Encapsulated Postscript and PNG. This functionality is also available via File->Export and via the toolbar.

freeze(message = None)

    Freezes further processing of the command line until the application receives an unfreeze via the GUI. The message argument, when supplied, is a string which is translated into a message dialog that is displayed to the user reminding him/her that a "freeze" command has been issued. If the message string is not supplied, no dialog is displayed before the command line is frozen. 
        Eg.
    freeze()             # Freezes the command line - no message dialog
    freeze("my message") # Freezes the command line & displays dialog
                         # containing the string, "my message"

get_data(index = -1)

    Returns either data for the indicated subplot or a list of all data plotted on the graph. Subplots are ordered starting with Y1 columns and continuing with Y2 columns.
    Data for each subplot is returned in a list like: 
        [[x1, x2, ..., xN], [y2, y2, ..., yN]]

get_figure()

    Returns the matplotlib figure object associated with the plotting area. The matplotlib figure is the entry point for all plotting functionality. To learn more about the plotting capabilities of matplotlib, visit the matplotlib website.

get_subplot(index = None)

    Returns either the indicated subplot object or a list of all subplot object contained in the figure object (when the index is not specified). The index argument is an integer which represents the subplot number within the figure object. Note that subplot indexing begins at zero. 
        Eg.
    get_subplot()   # Return a list of all subplot objects
    get_subplot(0)  # Returns the first subplot object
    To learn more about the plotting capabilities of matplotlib, visit the matplotlib website. subplot objects have all the properties of axes objects.

hold(b = None)

    Set the hold state. If hold is None (default), toggle the hold state. Else set the hold state to boolean value b. This controls both the matplotlib figure object and the GUI window. 
        Eg.
    hold()      # toggle hold
    hold(True)  # hold is on
    hold(False) # hold is off

open_file(file)

    Opens a previously saved session or user-created file containing DEAP commands. file is a string containing the path to the desired file.

redo()

    Redoes the last command typed in the interactive shell.

set_scale(index = 0, xlim = None, ylim = None)

    Rescale axes of given subplot to limits (xmin, xmax) and (ymin, ymax). If a set of limits is not given, it is set to autoscale.
    Returns the current limits.

undo()

    Undoes the last command typed in the interactive shell.

Plotting Panel Toolbar

The plotting panel allows user interactions with the plots. The toolbar simplifies access to the available interactive plot functions.

Open

    Displays the open file dialog and allows the user to open a previously saved session or to import a user-created file containing DEAP commands. This functionality is also available via File->Open.

Save

    Displays the save file dialog if a file has not been previously opened or saved and allows the user to save the current session to a new .txt file. If a file has been previously opened or saved, that file is automatically overwritten with the current command line history. This functionality is also available via File->Save.

Print

    Displays the print dialog and allows the user to print the figure displayed in the plotting panel. This functionality is also available via File->Print.

Export

    Displays the export file dialog and allows the user to save the figure displayed in the plotting panel to a file. Supported graphical formats are: Encapsulated Postscript and PNG. This functionality is also available via File->Export.

Unfreeze

    Unfreezes the processing of commands via the command line. This command is intended for use in conjunction with the freeze command. The freeze command is called at any point in a script to suspend further processing of script commands and allow the user to view/interact with the plot that has been created to that point. The Unfreeze button is invoked via the toolbar to resume operation. This functionality is also available via File->Unfreeze.

Undo

    Undoes your last command. If there are no commands to undo, the Undo command in the Edit menu will be unavailable and clicking the Undo toolbar button will result in no action. This functionality is also available via Edit->Undo.

Redo

    Redoes your last command. If there are no commands to redo, the Redo command in the Edit menu will be unavailable and clicking the Redo toolbar button will result in no action. This functionality is also available via Edit->Redo.

Unzoom

    Undoes a previously executed zoom. This functionality is also available via View->Unzoom.

Rezoom

    Redoes a previously executed zoom. This functionality is also available via View->Rezoom.

Info Tool

    Selecting the info tool allows the user to use the mouse pointer to focus in/out among the available subplots. Left-clicking the mouse brings a subplot into focus (hiding the other subplots). Right-clicking the mouse on the focused plot will show all subplots. If there is only one subplot, the info tool simply displays the mouse xy coordinates. This functionality is also available via Tools->Info.

Zoom Tool

    Selecting the zoom tool allows the user to use the mouse pointer for zooming in on a particular area of the plot. Left-clicking the mouse will zoom in. Right-clicking the mouse will zoom out. This functionality is also available via Tools->Zoom.

Pan Tool

    Selecting the pan tool allows the user to use the mouse pointer to pan around the selected subplot. Left-clicking the mouse and holding the left button down will pan around the subplot. Right-clicking restores the original view. This functionality is also available via Tools->Pan.

Grid Tool

    Selecting the grid tool turns on the plot grid. This functionality is also available via Tools->Grid.

Plot Edit Tool

    Selecting the plot edit tool allows the user to edit plot labels, colors, and title. Clicking on "Advanced Options" brings up an additional dialog which contains options for transparency, legend placement, and ordering of plots.
    Colors may be entered as hex codes or selected by clicking on the colored button to the right of the text field. Plots can only be reordered within their subplot - i.e., Y1 lines will always be below Y2 lines. Legend location can be specified with simple strings (e.g., "upper right") or coordinates 0-1 along the plot edges. If a string is chosen it will be used in place of any coordinates.
    This functionality is also available via Tools->Edit Plot.

User Manual

    The User Manual button displays the DEAP User Manual. This functionality is also available via Help->User Manual.

Menus

The menu bar found at the top of the application and contains many of the same commands as the toolbar with the addition of a few others.

While the icons provide quick, single-click access to application functions, the menu bar includes a categorized listing of the functions under menu options File, Edit, View, Tools, and Help.

File Menu

    Open
      Displays the open file dialog and allows the user to open a previously saved session or to import a user-created file containing DEAP commands. This functionality is also available in the toolbar.

    Save

      Displays the save file dialog if a file has not been previously opened or saved and allows the user to save the current session to a new .txt file. If a file has been previously opened or saved, that file is automatically overwritten with the current command line history. All commands typed into the command line are saved. This functionality is also available in the toolbar.

    Save As

      Displays the save file dialog and allows the user to save the current session to a new .txt file. All commands typed into the command line are saved. This functionality is also available in the toolbar.

    Page Setup

      Displays the page setup dialog. The page setup controls print settings.

    Print Preview

      Allows the user to see what the plot figure will look like when printed.

    Print

      Prints the plot figure. This functionality is also available in the toolbar.

    Export

      Displays the export file dialog and allows the user to save the figure displayed in the plotting panel to a file. Supported graphical formats are: Encapsulated Postscript and PNG. This functionality is also available in the toolbar.

    Unfreeze

      Unfreezes the processing of commands via the command line. This command is intended for use in conjunction with the freeze command. The freeze command is called at any point in a script to suspend further processing of script commands and allow the user to view/interact with the plot that has been created to that point. The Unfreeze button is invoked via the toolbar to resume operation. This functionality is also available in the toolbar.

    Exit

      Exits the application.

Edit Menu

    Undo

      Undoes your last command. If there are no commands to undo, this option will be unavailable. This functionality is also available in the toolbar.

    Redo

      Redoes your last command. If there are no commands to redo, this option will be unavailable. This functionality is also available in the toolbar.

    Clear

      Clears the plot figure. This is equivalent to calling clear() on the plot figure instance.

View Menu

    Rezoom

      Redoes a previously executed zoom. This functionality is also available in the toolbar.

    Unzoom

      Undoes a previously executed zoom. This functionality is also available in the toolbar.

    Display All Subplots

      Displays all subplots contained within a plot figure. This is equivalent to right-clicking on a focused subplot while using the info tool.

    Toolbar

      Toggles the display of the toolbar.

    Status Bar

      Toggles the display of the status bar at the bottom of the application.

    Full Screen

      Expands the application window to fit the entire screen. If already expanded, shrinks the application window back to the original size.
